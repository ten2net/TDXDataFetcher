# TDXDataFetcher 扩展计划

创建日期: 2026-04-10

---

## 项目现状

TDXDataFetcher (ten2net-tdxapi) 是一个通达信行情数据直连库，已实现：
- 实时行情、K线、分时、分笔数据
- 股票列表、财务数据、除权除息
- 公司信息、板块信息
- 期货行情支持

```bash
# 类型检查

black src/ tests/

```

## 设计原则

1. **向后兼容**: 所有新功能不破坏现有API
2. **可选依赖**: 高级功能使用 optional dependencies
3. **性能优先**: 批量操作使用批量协议请求
4. **类型安全**: 全代码类型注解
